Hacktiv8 Web Tugas 1

Saya mengikuti kelas sertifikasi ini untuk mengukur diri saya apakah ilmu yang telah saya pelajari sudah mendekati kelayakan sebagai programmer web pemula atau belum, sekaligus untuk belajar hal teknis maupun non-teknis yang mungkin saya lewatkan semasa belajar secara otodidak.

Saya percaya bahwa untuk menjadi programmer handal, kuncinya adalah rasa ingin tahu. Niat tersebut akan menstimulasi saya agar selalu mencari tahu kekurangan dalam diri serta memaksa diri saya sendiri untuk berkembang menjadi lebih baik demi menjadi programmer andal. Singkatnya, saya tidak bisa mengukur berapa lama waktu yang dibutuhkan untuk menjadi ahli, tetapi setidaknya saya membutuhkan kurang lebih satu bulan untuk memantapkan pengetahuan tentang fundamental pemrograman web melalui kelas ini.

Dengan mengikuti pelatihan ini, saya berharap dapat memperoleh ilmu baru dan mampu menerapkannya dalam proyek nyata sehingga dapat meningkatkan rasa percaya diri saya dalam melamar pekerjaan.
